## 🔧 **Getting Started**

### **Prerequisites**
- Node.js 18+
- Git
- UI-TARS API key

### **Installation**
```bash
# Clone the repository
git clone <your-repo-url>
cd ui-tars-workflow-hub

# Run setup script
npm run setup

# Start development server
npm run dev

# Start UI-TARS agent (in another terminal)
npm run agent
```

### **Configuration**
1. Copy `env.example` to `.env.local`
2. Add your UI-TARS API key
3. Configure your preferred AI model
4. Start the development server
